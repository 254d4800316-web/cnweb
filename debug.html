Nguyên văn đoạn mã bị lỗi (theo đề bài)
function addCourse(newCourse, list) {
    if (mssv % 2 !== 0) {
        list.push(newCourse);
    } else {
        list.unshift(newCourse);
    }
    renderTable(list);
}
Đoạn mã đã sửa + Chú thích
function addSkill(newSkill, list) {
    const mssv = 2544800316;   // MSSV của sinh viên

    if (mssv % 2 === 0) {      // Sửa điều kiện theo đề: chẵn thì push (cuối)
        list.push(newSkill);
    } else {
        list.unshift(newSkill); // Lẻ thì unshift (đầu)
    }
    renderTable(list);
}

Lỗi đã sửa:

  • Tên hàm sai: addCourse → addSkill
  • Điều kiện if bị ngược
  • Biến mssv chưa được khai báo
